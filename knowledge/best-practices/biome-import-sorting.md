# Biome インポートソート方針

## 方針

- **グループ順ソート**: 必要（外部モジュール → 内部エイリアス → 相対パス）
- **グループ内アルファベット順ソート**: 不要（開発中のノイズになり実用的価値が薄い）

## 現状の設定（Biome 2.x 時点）

Biome は「グループ順だけ強制し、グループ内は自由」という細かい制御をサポートしていない。
選択肢は以下の二択:

1. `organizeImports: "off"` — ソート強制を完全に無効化
2. デフォルト（on） — グループ順 + アルファベット順の両方を強制

方針に基づき **1** を採用。

```json
{
  "assist": {
    "actions": {
      "source": {
        "organizeImports": "off"
      }
    }
  }
}
```

## 将来の改善が入った場合

Biome が `organizeImports` にグループ順のみ強制するオプション（例: グループ間ソートは強制、グループ内ソートは無視）を追加した場合は、そちらに切り替える。

確認すべきポイント:
- Biome の Changelog / RFC で `organizeImports` の粒度設定が追加されたか
- `groups` オプションなどでグループ定義とグループ内ソートを分離できるか

該当機能が実装されたら `biome.json` を更新し、グループ順のみ強制する設定に移行すること。
